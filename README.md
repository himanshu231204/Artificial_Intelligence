- ✅ Hidden Markov Models (HMM)

## 🚀 How to Use This Repository

### For Complete Preparation (3-4 months)
1. **Week 1-2:** Search Algorithms
   - Start with uninformed search
   - Move to informed search
   - Practice adversarial search
   
2. **Week 3-4:** Logic and Reasoning
   - Master propositional logic
   - Understand predicate logic
   - Practice inference and resolution

3. **Week 5-8:** Probabilistic Reasoning
   - Build probability foundation
   - Understand Bayesian networks
   - Master inference methods
   - Study HMM thoroughly

4. **Week 9-12:** Practice and Revision
   - Solve all practice problems
   - Attempt previous year questions
   - Review quick revision notes

### For Quick Revision (1-2 weeks)
1. Go through Quick Revision folder
2. Practice numerical problems from each topic
3. Solve previous year questions
4. Review formulas and complexities

## 💡 Study Tips

### ✅ Do's
- **Practice numericals** - AI is calculation heavy
- **Code the algorithms** - Helps in understanding
- **Draw diagrams** - Especially for graphs and networks
- **Solve previous years** - Understand question patterns
- **Time yourself** - Practice under exam conditions

### ❌ Don'ts
- Don't just read theory - practice is key
- Don't skip probability - it's heavily weighted
- Don't memorize - understand the concepts
- Don't ignore edge cases in algorithms
